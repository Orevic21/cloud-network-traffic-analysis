# Cloud Network Traffic Analysis

## Overview

This project demonstrates network traffic analysis using Wireshark to investigate communication patterns, protocol usage, DNS activity, encrypted communications, and TCP connection establishment.
The objective was to develop foundational security monitoring skills that are applicable to Security Operations Center (SOC) environments and cloud security operations.

## Tools Used

- Wireshark
- Windows

## Environment

- Kali Linux
- VirtualBox

## Key Findings

- Captured 33,542 packets during a 10-minute 47-second session.
- TCP was the most frequently observed protocol.
- Identified 11,818 encrypted packets using TLS and QUIC.
- Observed DNS queries for domains including google.com and alphabot.app.
- Verified successful TCP three-way handshakes.
- Identified VLAN-tagged traffic generated by VirtualBox virtual networking.
